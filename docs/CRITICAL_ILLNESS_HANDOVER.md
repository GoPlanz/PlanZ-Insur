## 📊 数据结构

### 疾病数据类型

```typescript
interface Disease {
  id: string;
  name: string;
  nameEn: string;
  categoryId: DiseaseCategoryId;
  tier: 0 | 1 | 2 | 3 | 4;  // 0=早期, 1=高发, 2=核心, 3=老年, 4=常见
  description: string;
  keyConditions: string;
  claimsShare: string;
  severeCriteria?: string;      // 重症标准
  earlyCriteria?: string;       // 早期/轻症标准
  mainlandStandard?: string;    // 内地标准
  hongKongStandard?: string;    // 香港标准
}
```

### 五大分类

| ID | 名称 | 包含疾病 |
|----|------|---------|
| malignant | 恶性肿瘤（含原位癌） | 癌症、原位癌 |
| cardiovascular | 心脏及心脑血管疾病 | 心脏病、中风、搭桥、瓣膜、主动脉 |
| organ-failure | 器官功能衰竭与移植 | 肾衰竭、肝衰竭、呼吸衰竭、器官移植 |
| neurological | 神经系统与退行性疾病 | 脑肿瘤、阿尔茨海默、柏金逊、多发性硬化 |
| other | 其他严重疾病 | 失明、失聪、瘫痪、烧伤 |
