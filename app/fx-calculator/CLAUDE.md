# 汇率盈亏计算器 - 开发说明

## 概述
计算美元保单需要持有多少年才能抵消汇率下跌的损失

## 功能
- 输入：投保年份、汇率、保费、缴费年期、预期IRR
- 输出：汇率损失、累积收益、实际收益、盈亏平衡年
- 图表：汇率变化对收益的影响可视化

## 计算公式

### 汇率损失
```
汇率损失 = 保费总额 × (当前汇率 - 投保时汇率)
```

### 累积现金价值
```
现金价值 = Σ(每年保费 × (1 + IRR)^剩余年限)
```

### 盈亏平衡
当现金价值×当前汇率 - 汇率损失 > 0 时的年份
